# POTI Arredamenti - Modello Sito Web

Benvenuto nel modello di sito web professionale per POTI Arredamenti, l'azienda di arredamenti dal 1955.

## 📁 Struttura dei file

```
arredamento/
├── index.html       # Pagina principale HTML
├── style.css        # Stili CSS del sito
├── script.js        # JavaScript per interattività
├── logo.png         # Logo dell'azienda (da inserire)
└── README.md        # Questo file
```
## 🎯 Sezioni del sito

### 1. **Header/Navigazione**
- Logo POTI Arredamenti
- Menu di navigazione veloce
- Menu responsivo per dispositivi mobili

### 2. **Hero Section**
- Slogan principale: "Arredamenti di Qualità dal 1955"
- Pulsante per navigare al catalogo
- Design moderno con background gradiente blu/rosso

### 3. **Catalogo Prodotti**
- 6 categorie principali:
  - Soggiorni
  - Camere da Letto
  - Cucine
  - Uffici
  - Esterni
  - Accessori
- Card colorate e responsive

### 4. **Collezioni Speciali**
- Classico Elegante
- Moderno Minimalista
- Rustico Toscano

### 5. **Chi Siamo**
- Storia dell'azienda (dal 1955)
- Statistiche aziendali
- 4 vantaggi principali con icone

### 6. **Contatti**
- Indirizzo
- Telefono
- Email

- Form di contatto

### 7. **Footer**
- Link social media
- Categorie prodotti

## 🎨 Colori utilizzati

- **Blu Navy (Primario)**: `#001f4d` - Colore principale del logo
- **Rosso/Arancione (Secondario)**: `#ff4444` - Accenti e dettagli
- **Chiaro**: `#f8f9fa` - Sfondo sezioni alternate
- **Scuro**: `#1a1a1a` - Testo principale

## 📱 Responsive Design

Il sito è completamente responsive e si adatta perfettamente a:
- 📱 Dispositivi mobili (480px+)
- 📱 Tablet (768px+)
- 💻 Desktop (1200px+)

## ⚙️ Funzionalità JavaScript

- **Smooth scrolling** per i link di navigazione
- **Mobile menu** hamburger per dispositivi piccoli
- **Form validation** per il modulo di contatti
- **Scroll animations** con fade-in dei card
- **Indicatore di sezione attiva** nella navigazione
