# Luxury Arredi - Template Arredamento

Un template moderno, performante e completamente responsive per siti web di arredamento e interior design.

## Caratteristiche principali
- **Performance**: Ottimizzazione del caricamento immagini (Lazy loading e Pre-loading).
- **Responsive**: Design adattivo con gestione avanzata del menu mobile.
- **UX/UI**: Header dinamico allo scroll, animazioni fade-in e navigazione fluida.
- **Branding**: Completamente anonimizzato e pronto per l'uso come template.

## Tecnologie utilizzate
- HTML5 & CSS3 (Custom Properties e Grid/Flexbox)
- JavaScript Vanilla (Intersection Observer API per le animazioni)
- Font Awesome & Google Fonts
